# Personal Claude Configuration
This repository contains my personal Claude Code configuration for web VMs.
## Contents
- `skills/` - Custom Claude skills available in all projects
- `docs/global-claude.md` - Global instructions and preferences
- `bash/tools.sh` - Bash tools and aliases
- `settings.json` - Claude Code settings (incl. status line)
## Usage
This repo is automatically cloned by SessionStart hooks in my projects.
See `rplsmn/claude-project-template` for the hook implementation.
